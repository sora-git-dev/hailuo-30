# Hailuo 3.0 Usage Guide

## Online Access (Recommended)

Visit **[hailuo30.net](https://hailuo30.net/)** for the easiest experience.

## Text-to-Video

1. Enter your text prompt
2. Select resolution (720p / 1080p / 4K)
3. Click "Generate"

### Example Prompts

```
A majestic eagle soaring through mountain clouds at golden hour
```

## Image-to-Video

1. Upload your source image
2. Add motion description (optional)
3. Generate
