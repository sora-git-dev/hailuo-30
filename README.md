## ✨ Features

[Hailuo 3.0](https://hailuo30.net/) is a next-generation AI video generator that creates stunning cinematic videos with smooth motion and professional quality. It supports both text-to-video and image-to-video generation in up to 4K resolution.

## 🚀 Quick Start

### Online Access (Recommended)

The easiest way to experience Hailuo 3.0:

1. Visit **[hailuo30.net](https://hailuo30.net/)**
2. Enter your text prompt or upload an image
3. Select video style and resolution
4. Click "Generate" and watch the magic happen!

```
No installation required • Free tier available • 4K output supported
```
